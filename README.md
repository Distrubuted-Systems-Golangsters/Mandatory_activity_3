## How to run the program:
1. **Clone the Repository:**
   git clone https://github.com/yourusername/chitty-chat.git
   cd chitty-chat

2. **Open the server (In terminal):**
- cd .\server\
- go run .\main.go

3. **Add a client (In a different terminal):**
- cd .\client\
- go run .\main.go
